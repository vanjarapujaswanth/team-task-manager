# Team Task Manager

A modern full-stack Team Task Manager web application built using Flask, SQLAlchemy, HTML, CSS, and JavaScript. The application allows Admins and Members to manage projects, assign tasks, track progress, and monitor overdue work with role-based access control.

---

# Features

## Authentication
- User Signup
- User Login
- Secure Password Hashing
- Session Management

## Role-Based Access
- Admin Role
- Member Role

## Project Management
- Create Projects
- View All Projects
- Delete Projects (Admin Only)

## Task Management
- Create Tasks
- Assign Tasks to Members
- Update Task Status
- Delete Tasks (Admin Only)
- Task Search & Filtering

## Dashboard
- Total Tasks
- Completed Tasks
- Pending Tasks
- Overdue Tasks
- Progress Tracking

## UI Features
- Responsive Design
- Modern SaaS Style UI
- Dark / Light Theme Toggle
- Glassmorphism Design
- Animations & Hover Effects
- Color-Coded Task Status
